# AI Text Summarizer

## Use Case
Completed the Text Summarization use case from Nasscom AI Code Sarathi.

## Features
- Summarizes long text
- Multiple summary lengths (short, balanced, detailed)
- Bullet point summaries
- Automatic title generation

## Tech Stack
- Python
- Streamlit
- HuggingFace Transformers

## How to Run
pip install -r requirements.txt  
streamlit run app.py
